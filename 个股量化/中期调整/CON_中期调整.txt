{ -----------------------------------------------------
技术要点：
    当100日或200日均线处于多头状态时，选择100日周期
    否则默认使用200日周期
    仅限日线周期生效（PERIOD=5）

    可以增加成交量验证：VOL>MA(VOL,20)*1.5，避免均线假突破
----------------------------------------------------- }
N :   IF( (MA多(100) || MA多(200))  AND  PERIOD=5{日},     100,     200)       NODRAW;


{ -----------------------------------------------------
计算V字形指标：
    N日内最高价
    最高价出现位置

    最高点后的最低价
    最低价出现位置
----------------------------------------------------- }
_H    :   HHV(H,     N);
H_DAY :   HHVBARS(H, N)      +1;

_L    :   LLV(L,     H_DAY);
L_DAY :   LLVBARS(L, H_DAY)  +1;


{ -----------------------------------------------------
指标类型	    计算基准	        应用场景
调整幅度	    历史最大回撤	    风险控制参考
调整幅度2	当前相对跌幅	    超跌反弹机会识别
调整天数	    最大回撤持续时间	趋势强度评估
调整天数2	最高点距今时间	    顶部压力评估
----------------------------------------------------- }
{最低价的调整幅度和天数}
调整幅度  :  _L/_H *100-100;
调整天数  :  H_DAY - L_DAY ;

{当前价的调整和天数}
调整幅度2 :  L/_H *100 -100;
调整天数2 :  H_DAY -1      ;
